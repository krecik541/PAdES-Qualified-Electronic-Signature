Detailed Description

The Sign class is responsible for signing PDF documents using an encrypted private key and the SHA256withRSA algorithm.

void org.example.developed_app.sign.Sign.init ( ) throws Exception

Initializes the process of signing a PDF document.

Loads the private key from an encrypted file, decrypts it using the PIN, calculates the hash of the PDF content, signs it, and saves the signature in the PDF metadata.

Exceptions
ExceptionIf an error occurs during decryption, signing, or saving the document.
